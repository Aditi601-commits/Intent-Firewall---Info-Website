1. What is IntentFirewall?

IntentFirewall is an innovative mobile security application that protects users from sophisticated social engineering attacks and deepfake audio threats. Built with cutting-edge machine learning and multi-layered defense systems, it represents the next generation of on-device security.

2. The Vision Behind the Project

Created as part of the Hackaccino initiative, IntentFirewall tackles the growing threat of synthetic media and advanced social engineering. The app combines traditional security approaches with modern AI to create a comprehensive defense system.

Core Mission

Stop scam messages and detect deepfake calls before they can cause harm

Privacy-First

All processing happens locally on your device - no data leaves your phone

Multi-Tier Defense

Combines regex filtering, text classification, and audio analysis

3. How It Works

The system is built across multiple platforms with specialized components for different types of threats.

Mobile App

React Native interface with native Android integration for real-time threat detection

Text Analysis

Multi-stage text classification using distilled BERT models and heuristic scoring

Audio Deepfake Detection

TFLite models analyze call audio using LFCC features and WavLM embeddings

ML Pipeline

Python-based training and distillation system for model optimization

4. Technology Stack

IntentFirewall leverages modern technologies to deliver high-performance security:

Frontend & Mobile

  • React Native: Cross-platform mobile development
  • Android Native: Direct system integration for notifications and calls
  • AsyncStorage: Local data persistence for settings and threat history

AI & Machine Learning

  • TensorFlow Lite: On-device inference for real-time analysis
  • DistilBERT: Compact text classification models
  • WavLM: Advanced audio feature extraction
  • Custom ML Pipeline: Python-based training and model optimization

5. Key Features

The app provides comprehensive protection with user-friendly controls:

  • Real-time Message Scanning: Automatically analyzes incoming notifications
  • Voice Call Protection: Detects synthetic speech during phone calls
  • Threat History: Complete log of detected and blocked threats
  • Customizable Settings: Adjust sensitivity and notification preferences
  • Offline Operation: Works without internet connectivity

6. What's Next

The project continues to evolve with advanced threat detection capabilities:

  • Enhanced audio deepfake detection algorithms
  • Expanded language support for global protection
  • Integration with additional messaging platforms
  • Advanced federated learning for threat intelligence
